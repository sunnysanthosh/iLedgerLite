# iLedgerLite Branding Assets

This folder contains the visual identity and marketing collateral for iLedgerLite.

## 📁 Files

- **iledgerlite_logo_slanted_l_...png**: The primary logo with the modern slanted 'L'.
- **iledgerlite_app_banner_refined_...png**: High-resolution marketing banner.
- **iledgerlite_social_media_post_refined_...png**: Social media graphic for launches.
- **icons/*.svg**: 9 core SVG icons (Income, Expense, Account, Report, Sync, Notification, Settings, AI, App Icon).
- **Originals**: I have also included the first design iterations for your reference.

## 🎨 Specifications

### Colors
- **Navy**: `#1B2D4B`
- **Orange**: `#E68A2E`
- **Medium Blue**: `#2B6BAE`

### Typography
Recommended: **Inter** or **Outfit** (Sans-serif).

### Icons
We recommend the [Lucide](https://lucide.dev/) icon set for a clean, cohesive mobile and web experience.

## 🛠️ Note on File Formats
These are high-resolution PNG (raster) images. While they can be scaled down or used in digital mockups, they are not "vector" files (like SVG or AI). For permanent brand extension (e.g., massive billboards), a vector version of the logo would be the next step.
